# 🌸 Iris Flower Classification using Random Forest

This project demonstrates how to build a **Machine Learning classification model** using the **Random Forest Classifier** from Scikit-learn. The model is trained on the famous **Iris dataset** downloaded automatically using **KaggleHub**.

## 📌 Project Overview

The program performs the following steps:

1. Downloads the Iris dataset using KaggleHub.
2. Loads the dataset with Pandas.
3. Selects the feature columns and target labels.
4. Splits the dataset into training and testing sets.
5. Trains a Random Forest Classifier.
6. Makes predictions on the test data.
7. Evaluates the model using:
   - Accuracy Score
   - Classification Report
   - Confusion Matrix

## 🌺 Dataset

**Dataset:** Iris Dataset

The dataset contains **150 flower samples** from three Iris species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

Each sample includes four features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
